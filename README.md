# Arq & Urb — Plataforma Digital

> **Projetando ideias, transformando espaços.**

Site institucional e portfólio digital da agência acadêmica **Arq & Urb**, desenvolvido como projeto das disciplinas de Criação e Arte Publicitária, Arquitetura e Urbanismo.

---

## 🚀 Deploy na Vercel

### Opção 1 — Via Vercel CLI

```bash
npm install -g vercel
vercel --prod
```

### Opção 2 — Via Dashboard (recomendado)

1. Acesse [vercel.com](https://vercel.com) e faça login
2. Clique em **"Add New Project"**
3. Importe o repositório Git do projeto (ou faça upload da pasta)
4. As configurações serão detectadas automaticamente via `vercel.json`
5. Clique em **"Deploy"** ✅
